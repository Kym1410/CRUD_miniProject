# CRUD_miniProject

## miniproject주제 설명
저희 project는 대학생을 대상으로 한 시간표 작성 프로그램입니다.

## 기능 설명
학생이 수강하는 과목의 정보(과목명, 요일, 시간)을 입력받고 관리 및 저장하는 프로그램입니다.
학생이 수강하는 과목을 입력하면 이를 시간표의 형태로 출력해주고 각 시간(교시), 요일, 과목명 별로 검색할 수 있는 기능과 시간표 저장 및 조회 기능이 들어가있습니다.
